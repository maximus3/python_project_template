## Commands

### Docker clean, build, up, clean
    make docker

### Up docker container:
    make docker-up

### Down docker container:
    make docker-down

### Create venv (if no docker):
    make venv

### Run app:
    make up

### Run tests:
    make test

### Run linters:
    make lint

### Run formatters:
    make format

### Run format and lint code then run tests:
    make check
